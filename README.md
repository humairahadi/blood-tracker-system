## Project Description
The Blood Tracker System is a web-based application developed as an pairing project for the IMS566 (Advanced Web Design Development and Content Management) course. The system is designed to help manage and track and requests blood donation and blood availability. This project demonstrates the application of modern web design principles, responsive layout, and effective content management through a functional web application interface.

## Features
- Login and authentication system using hardcoded credentials
- Responsive navigation menu available on all pages
- Dashboard page displaying summary information related to total blood unit, registered donors amd current blood stock status
- Data view pages for blood analytics, blood donation requests and blood stock inventory
- Data visualisation using charts and graphs to represent donation statistics or distribution stock
- Responsive and mobile-friendly user interface using a modern front-end framework
