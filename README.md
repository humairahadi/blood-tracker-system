## Project Description
The Blood Tracker System is a web-based application developed as an pairing project for the IMS566 (Advanced Web Design Development and Content Management) course. The system allows users to view a dashboard overview, view blood analysis, submit blood donation requests, check blood stock status and access individual student e-profiles. Developed using a front-end framework, the interface emphasizes usability, consistency and responsive design while demonstrating practical application of advanced web design principles.

## Features
- Login and authentication system using hardcoded credentials
- Responsive navigation menu available on all pages
- Dashboard page displaying summary information related to student e-profiles, total blood unit, registered donors and current blood stock status
- Data view pages for blood analytics, student e-profiles, blood donation requests and blood stock inventory
- Data visualisation using charts and graphs to represent donation statistics or distribution stock
- Responsive and mobile-friendly user interface using a modern front-end framework
